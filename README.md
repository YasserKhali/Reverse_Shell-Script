# Reverse_Shell-Script
Demonstration for Security Research (EXAMPLE — Educational Use Only)

Important — ethical and legal notice
This repository contains material intended strictly for educational and research purposes. None of the resources here should be used to harm, evade protections, or attack systems. All tests must be run only in an isolated environment (virtual machines without public network access, lab networks under control) and with proper authorization.

Goal:
This repository document a security research / experiment aimed at understanding how certain behaviors are detected by antivirus and detection tools. The objective is educational: to improve detection and defense practices, not to provide evasion methods.
